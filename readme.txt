Проект содержит тест checkTitle():
он открывает главную страницу otus.ru и проверяет title страницы.

Адрес проверяемой страницы задан в файле 'config.properties'.
Интерфейс 'TestConfig' с использованием библиотеки Owner импортирует значение адреса по ключу 'url'.
В классе CheckTitleTest содержится основной код - настройка драйвера и тестовый сценарий.

Проект использует: Selenium Java, WebDriverManager, jUnit, Owner, log4j.
